Poner en funcionamiento el backend:
Abrir la solución en Visual Studio y darle a ejecución al botón de http.

Verificar el inicio:
Cuando verifiquemos que el Swagger se inicia correctamente en el navegador, pasamos al siguiente paso.

Ejecutar el frontend:
Abrir la terminal de VS Code (dentro de la carpeta frontend) y ejecutar el siguiente comando:

npm run dev


Acceso:
Por último, veremos que salta en el navegador la pantalla de login de nuestro trabajo.
